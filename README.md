# BurgerKiosk

# (C# 코딩) 버거 키오스크
## 개요

- C# 프로그래밍학습

- 1줄소개: 사용자의 선택을 입력 받아 처리하는 프로그램

-사용한플랫폼: 
	- C#, .NET Windows Forms, Visual Studio, GitHub

-사용한컨트롤:
    - Label, ListBox,, Button, GroupBox, RadioButton, CheckBox
